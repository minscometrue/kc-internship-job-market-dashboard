# KC Internship & Job Market Intelligence Dashboard

"A Python dashboard project that collects and analyzes entry-level and internship job postings for Python, Data, AI, and Software roles in the Kansas City area."


## Project Goal
"The goal of this project is to analyze the following insights based on the collected job posting data"

* What tech stacks are most frequently required?
* Which companies are hiring Python, Data, AI, and Software interns?
* Which industries have the most job openings?
* How well does my current skill set align with each job posting?
* Which job postings should I prioritize applying?


## MVP Features
- Manual CSV job posting input
- Skill frequency analysis with pandas
- Streamlit dashboard
- Job-to-skill matching score


## Tech Stack
- Python
- pandas
- Streamlit
- matplotlib
- Git/GitHub
